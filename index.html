Rafi Putra Ramadhan

Tangerang Selatan, Indonesia
Mahasiswa - Web Developer
University of Bina Sarana Informatika

Perkenalkan nama saya Rafi Putra Ramadhan biasa dipanggil Rafi, berusia 21 tahun. Saya lahir di Jakarta pada tanggal 11 Desember 2000. Saat ini saya tinggal di Komplek Vila Inti Persada, kec. Pamulang, kel. Pamulang Timur, Kota Tangerang Selatan. Pendidikan yang sedang saya tempuh saat ini yaitu, kuliah. Saya kuliah di Universitas Bina Sarana Informatika, program studi Teknologi Informasi dan saat ini sedang berada di Semester 6.

Saya adalah orang yang sangat tertarik dengan dunia Teknologi, sebelumn terjun ke dunia Programming, pada saat SMK saya mengambil jurusan Multimedia, saya suka membuat hal kreatif seperti membuat design photo, design poster/banner, membuat shortmovie, dan lain-lain. dan pada kelas 2 SMK saya diterima Praktik Kerja Lapangan (PKL) di salah satu instansi pemerintah yaitu Pustekkom Kemdikbud Ciputat sekarang berubah menjadi Pusdatin Kemdikbud. Di Pustekkom saya masuk bagian RTF sebagai Content Creator untuk platform yang diberi nama Ensiklomedia. Ensiklomedia merupakan platform edukasi digital yang berisi banyak video edukasi seperti tutorial dan lain-lain. Saya diwajibkan merilis setidaknya 1 video dalam 1 minggu, dan di sana saya belajar bagaimana cara membuat content seperti penulisan script, voice over, dan lain-lain.

Saya mulai tertarik pada dunia programming pada saat PKL tersebut, karena pada saat itu saya pernah mendapat tawaran itu mendesign, tetapi tidak menggunakan photoshop, illustrator atau pun yg lain, tetapi menggunakan CSS untuk layouting beberapa design website. Karena saya baru mendengarnya jadi saya blm bisa menerima tawaran tersebut, tetapi karena ada rasanya penasaran saya mencari tahu lebih lanjut apa itu dan kegunaannya untuk apa dan sebagainya, saya pun menemukan hal menarik yaitu tentang Web Development. Saya pun banyak mengulik tentang Web Development tersebut dan mengikut Ekstrakurikuler membuat Web serta mengikuti kursus untuk Web Development. Bahasa pertama yang pelajari yaitu C++ pada awal semester 1 di kampus, dan lanjut ke Java, pada pemrograman Java saya mencoba ke ranah GUI, dan Website. Framework Java yang saya gunakan untuk membangun website yaitu Spring Boot, dan project pertama yang saya buat yaitu Sistem RT yang dipegang oleh teman saya walaupun saya lebih banyak untuk mengubah template yang sudah. Namun, karena tidak adanya perkembangan dalam diri saya saat mempelajari Spring Boot, saya mencoba Framework lain untuk membangun website yaitu Laravel. Saya memilih Laravel karena resource pembelajaran yang mudah didapatkan dan studi kasus lebih mudah dipahami dan karena hal tersebut yang membuat saya makin nyaman dalam mempelajari dan memperdalami skill saya dibidang Fullstack Web Development.
